- Create React App Xtras


# Using
- Create a React app and add CRAX
  ```
  $ npx create-react-app my-app
  $ npm i -D crax
  $ crax init
  ```
  - This will creat a `.crax` file in the root of your application

- Package.json script
  ```json
    "scripts": {
      "new:component": "crax component"
    },
  ```
